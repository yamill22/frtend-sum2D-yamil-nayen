# TechFix - Página Web

## Descripción
TechFix es una empresa profesional dedicada a la reparación y mantenimiento de dispositivos electrónicos. Este proyecto corresponde a la **Etapa 1**, donde se ha desarrollado la estructura básica de la página web sin estilos avanzados.

## Secciones Implementadas
1. **Barra de Navegación (Navbar)**:
   - Incluye el nombre/logo de la empresa y enlaces a las secciones principales: Inicio, Servicios, Precios, Nosotros e Ingresar equipo.

2. **Sección Hero**:
   - Imagen de fondo relacionada con tecnología.
   - Título principal con el nombre del servicio técnico.
   - Slogan que transmite confianza y rapidez.
   - Botón principal para ingresar equipos.

3. **Indicadores de Confianza**:
   - Tres indicadores visuales: Diagnóstico gratuito, Garantía escrita y Más de 8 años de experiencia.

4. **Sección de Servicios**:
   - Tarjetas que muestran los servicios ofrecidos, como reparación de notebooks, cambio de pantalla, limpieza, recuperación de datos, entre otros.

5. **Sección de Marcas y Dispositivos Atendidos**:
   - Logos y nombres de marcas reconocidas como HP, Dell, Lenovo, Apple, Samsung, Asus, Acer, Canon y Epson.

6. **Sección de Testimonios**:
   - Comentarios de clientes ficticios con el servicio recibido, texto del testimonio y calificación.

7. **Footer**:
   - Información de contacto, horarios de atención, redes sociales y derechos de autor.

## Estilos Implementados (style.css)
En esta etapa, se definieron los estilos básicos para la página web, incluyendo:

1. **Paleta de Colores**:
   - Se utilizaron colores que transmiten tecnología y confianza: azul oscuro, gris claro, blanco y un color de acento (azul brillante). Estos colores se definieron como variables CSS para facilitar su uso.

2. **Tipografías**:
   - Se implementaron dos tipografías de Google Fonts:
     - `Montserrat` para los títulos, con un estilo moderno.
     - `Roboto` para el cuerpo del texto, priorizando la legibilidad.

3. **Efectos en Tarjetas**:
   - Las tarjetas de servicios tienen un efecto hover con una transición suave, que incluye elevación y sombra.

4. **Tabla de Precios**:
   - Se diseñó con una cabecera de color, filas alternadas con fondo diferente y bordes definidos para mejorar la legibilidad.

5. **Diseño Responsivo**:
   - El sitio se adapta a pantallas móviles:
     - Las secciones de servicios y testimonios se apilan en una sola columna.
     - La tabla de precios es desplazable horizontalmente si no cabe en la pantalla.

Estos estilos aseguran un diseño moderno, funcional y adaptable a diferentes dispositivos.
